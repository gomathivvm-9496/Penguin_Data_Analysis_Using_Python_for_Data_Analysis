# 🐧 Penguins Data Analysis Project

## 📌 Overview
This project performs data analysis on a penguin dataset to understand patterns in species, body mass, and physical characteristics. It includes data cleaning, visualization, and basic web scraping integration.



## 📊 Dataset Description
The dataset contains information about penguins, including:
- Species
- Culmen Length & Depth
- Flipper Length
- Body Mass
- Sex
- Region & Island



## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- BeautifulSoup (Web Scraping)



## 🔧 Data Cleaning Steps
- Removed missing values using `dropna()` and `fillna()`
- Cleaned column names (removed spaces and special characters)
- Converted date column to datetime format
- Standardized species names



## 🧠 Feature Engineering
- Created a new column **Size** based on Body Mass:
  - Small
  - Medium
  - Large



## 📈 Visualizations
- Scatter Plot (Flipper Length vs Body Mass)
- Line Chart (Body Mass Trend)
- Bar Chart (Species Distribution)
- Pie Chart (Species Proportion)
- Histogram (Body Mass Distribution)



## 🔍 Key Insights
- Gentoo penguins have higher body mass compared to other species
- Strong positive relationship between flipper length and body mass
- Adelie penguins are the most common species
- Body mass distribution is slightly right-skewed



## 📊 Analysis Types
- **Descriptive**: Summarized species distribution and measurements
- **Diagnostic**: Identified relationships between features
- **Predictive**: Estimated body mass using flipper length
- **Prescriptive**: Suggested improvements and analysis strategies



## 💡 Recommendations
- Use flipper length as a key predictor for body mass
- Focus on species-based analysis
- Handle missing values and outliers carefully
- Collect more environmental data for deeper insights



## 📚 Learnings
- Data cleaning and preprocessing
- Data visualization techniques
- Feature engineering
- Web scraping integration
- Analytical thinking (4 types of analysis)


## 🚀 Conclusion
This project demonstrates a complete data analysis workflow, including data cleaning, visualization, feature engineering, and web scraping integration.

The analysis highlights key patterns in penguin species, especially the relationship between flipper length and body mass. Overall, the project shows how data-driven insights can be used to understand real-world data effectively.
